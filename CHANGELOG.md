# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2025-03-24

### Added
- Initial public release
- Rogue AP detection using whitelist-based approach
- Active scanning mode via `iwlist`
- Passive scanning mode via `scapy` packet sniffing
- RSSI signal strength analysis
- Configurable alert threshold
- GPS integration support
- JSON output for detected access points
- Whitelist management via JSON configuration
- Comprehensive logging
